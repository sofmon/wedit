What is wedit.io?

Seeing is believing: Press the 'ctrl' key and edit this page

Wedit is a static site generation tool that is using the web browser as a WYSIWYG editor based on user defined HTML template.

Wedit is an Open Source project under MIT license. We believe you have the right to know what is running on your machines.

You are more than welcome to contribute. Fork us on github and get wild.

How can I use it?

Just get the Wedit tool on your MAC, Windows or Linux machine

You can install Wedit tool using:

  • Brew for MAC: brew install wedit
  • Scoop for Windows: scoop install wedit
  • Even though we don't recommend it, you can get [direct binaries] for your operating system
  • Or compile your own version from our Git repository using Go and Dart

The tool is statically compiled so once you have it working there is no needs for any external dependencies.

How it works?

Get a beautiful  static HTML  website and turn it into a template

Setup solution

  • Ensure you have 'wedit' tool installed on your working environment
  • Create a working folder: <site>
  • Within the working folder run: wedit init
  • Put your HTML template into folder: <site>/template/
  • Modify the template by adding attributes: data-var

Template structure

  • Each page in your template must have own sub-folder that is to be mapped to the request URL
  • All HTML templates must be located within the corresponding sub-folders and be named index.html
For example URL: http://<your_site>/something/here will hit template <site>/something/here/index.html

data-var= 'key '

  • In your template index.html add data-var= '<key> ' attribute to any item you want to be able to edit.
  • The <key> must be unique for the page.

data-var-repeat= '<key> '

  • In your template index.html add data-var-repeat= 'key ' for all items you want to be repeatable.
  • The <key> must be unique for the page
